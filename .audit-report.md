### P1: Manager.Steer() 阻塞发送可能死锁 (§5.2)
```go
func (m *Manager) Steer(jobID string, message string) error {
    job, ok := m.jobs[jobID]
```

父 Agent 的 `Steer()` 使用 `select` + `default` 非阻塞发送。Manager 的 `Steer()` 也应一致。
